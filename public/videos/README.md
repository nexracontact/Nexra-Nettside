# Videos-mappen

Legg video-filer her som skal brukes i prosjekter.

## Støttede formater
- MP4 (anbefalt)
- WebM
- MOV

## Eksempel
Hvis du legger en fil som `prosjekt1.mp4` her, kan du bruke den i Portfolio.tsx slik:

```tsx
video: {
  src: '/videos/prosjekt1.mp4',
  poster: '/images/portfolio/prosjekt1.jpg',
}
```

## Tips
- Komprimer videoer før opplasting (maks 5-10MB anbefalt)
- Bruk MP4-format for best kompatibilitet
- For større videoer, vurder å laste opp til YouTube eller Vimeo i stedet
